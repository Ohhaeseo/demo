## 📌 프로젝트 소개

이 프로젝트는 다음과 같은 기능을 제공합니다:
- **개인 프로필**: 자기소개와 연락 정보.
- **프로젝트 소개**: 내가 수행한 주요 프로젝트들을 나열하여 보여줍니다.
- **스킬 리스트**: 내가 가진 기술 스택을 보기 좋게 정리.
- **게시판**: 블로그 포스팅 또는 사용자 피드백을 받을 수 있는 게시판 기능.
- **파일 업로드 및 관리**: 파일을 업로드하고 관리할 수 있는 시스템.
